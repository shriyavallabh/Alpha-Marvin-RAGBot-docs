# JurisAgent Documentation

This directory contains comprehensive documentation for the JurisAgent legal RAG system.

## Documentation Structure

- **Product**: Feature descriptions, roadmaps, use cases
- **Architecture**: System design, deployment, API specifications  
- **Administration**: User management, dashboards, role definitions
- **Market**: Competitive analysis and differentiators
- **Operations**: RACI matrix, timelines, responsibilities

## Viewing the Documentation

The documentation is built with MkDocs Material and deployed via GitHub Actions.

Live site: https://shriyavallabh.github.io/Alpha-Marvin-RAGBot/

## Building Locally

```bash
pip install mkdocs-material mkdocs-mermaid2-plugin
mkdocs serve
```

Then open http://localhost:8000
